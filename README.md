# Velora



## Описание

Velora - это телеграм бот для вывода расписания зарегистрированных колледжей. Это проект, который был создан для обучения и экспериментов с библиотекой [aiogram](https://docs.aiogram.dev/en/latest/).

## Для чего это нужно?

Благодоря этому проекту я смог изучить основы работы с библиотекой aiogram, а также научился создавать ботов для Telegram. Этот бот может быть полезен для студентов и преподавателей колледжей, которые хотят быстро получить информацию о расписании занятий.

Так же тут реализована структура бизнес логики, которая может быть использована в других проектах. Ниже описана структура проекта и его основные компоненты.

## Структура проекта

Корневая структура

```
├───data
├───db
├───docs
├───logs
└───src
```

`data/` - Различные данные

`db/` - Локальная база данных, то есть если у нас не подключен Postgresql

`docs/` - Документация проекта

`logs/` - Хранилище логов

`src/` - Бизнес-логика и архитектура бота

```
└───src
    ├───config
    ├───core
    ├───filters
    ├───handlers
    ├───middlewares
    ├───models
    ├───services
    └───utils
```

`config/` - Конфигурации: токены, ключи, пути, настройки базы.

`core/` - Инициализации приложения (использован app_context для удобной работы с зависимостями)

`filters/` - Пользовательские фильтры aiogram (например, IsAdmin)

`handlers/` - Все хендлеры: команды, сообщения, колбэки

`middlewares/` - Middleware для логирования, rate-limit и т.д.

`models/` - Модели базы данных

`services/` - Сервисы

`utils/` - Разные вспомогательные функции, (например, генерация уникального кода товара)
